# Eolicplants - Server

### Servidor implementado en Node.js
___

### Requerimientos

    - Tener instalado y ejecuntandose el servicio de MySql (en el puerto 3306)
    - Ejecutar el script de creacion de la base de datos "scriptBD.sql"
    - Arrancar el servidor de Node con "npm start" o con "node app.js"
